# STM32F4-MPU6050-I2C-with-Timer
- Sensor GY-521 6DOF IMU MPU6050 interface STM32F411 using I2C. - Create Timer 2 generate delay 1ms to update data roll, pitch, yaw or titl angle for Balance Robot
